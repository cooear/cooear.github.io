XX-Net接力GoAgent的工具

2016-05-04 8:46 pm

XX-Net是一个接力GoAgent的翻墙工具，提供GAE proxy并且稳定、易用、快速，Web界面，人性化交互等是大家能够科学上网。

XX-Net可以当作是一个已经配置好的goagent（目前在Github上托管），并集成了gogotester，那么总的来说有：

XX-Net的基本特点

  • 集成搜索Google IP；
  • 基于Web 的配置界面、上传部署、Log查看，也就是说你的所有配置都可以在网页上进行，比较直观；
  • 内置了公共 appid, 安装即可上网, 当然菊部哥依然推荐注册自己的appid（如何注册Google Appid）；
  • 新版本提示、更新升级。

XX-Net支持平台

  • Windows XP （需要 tcpip.sys 补丁, 比如用tcp-z）
  • Windows 7/8
  • Ubuntu （不能显示系统托盘）
  • Debian
  • Mac OS X
